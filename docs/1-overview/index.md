Overview
======================

실시간 서비스란 무엇이며, 어떻게 활용될 수 있는지, 그리고 향후 본 프로젝트가 어떻게 발전하게 될 것인지 설명합니다. 
전반적인 실시간 서비스 시장에 대해 알아보고, 본 프로젝트의 필요성과 발전 가능성에 대해 설명하고자 합니다.

본 프로젝트를 상세히 이해하기에 앞서 실시간 서비스 시장에 대한 이해가 필요하며, 기술적인 설명을 제외하여 개발자가 아니어도 실시간 서비스에 대한 이해가 가능하도록 구성하였습니다.
